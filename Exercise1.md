# 🎯 SQL Bolt Exercises

🚀 A beginner-friendly repository to master **SQL fundamentals** using hands-on exercises from SQLBolt.

---

## 📌 About This Project

This repository contains solutions to SQLBolt exercises, covering:

* SELECT queries
* Data retrieval
* Multiple column selection
* Real dataset practice (Movies table)

Perfect for:

* 🧑‍🎓 Students
* 💻 Beginners learning SQL
* 🎯 Interview preparation
